**Startup Funding Analysis**
**Overview**

This project analyzes Indian startup funding using a cleaned Excel dataset and a Power BI dashboard. The goal is to understand which cities and industries receive the most funding and to highlight overall funding trends.

**Files Included**

Indian Startup Funding Cleaned.xlsx – Cleaned dataset prepared for analysis

Indian Startup Funding Power BI Dashboard.pbix – Power BI dashboard file

Indian Startup Funding Power BI Dashboard.png – Screenshot of the Power BI dashboard

**Data Cleaning (Excel)**

The dataset was cleaned to make it consistent and ready for visualization. The steps included:

Standardizing column names

Removing duplicates

Cleaning and unifying city names

Extracting the year from the date

Converting the funding amount to numeric format

Reordering columns for better structure

The cleaned file is the final version used in Power BI.

**Power BI Dashboard**

The dashboard presents key insights from the cleaned dataset. It includes:

KPI cards showing total funding, number of startups, investors, and funded cities

Top 10 funded industries

Funding amount by city

A clear and simple layout for easy understanding

**How to Use**

Download the Excel file to view the cleaned dataset.

Download the .pbix file and open it in Power BI Desktop.

Explore the visuals and interact with filters if needed.

**Purpose**

This project demonstrates the process of preparing real-world data and turning it into meaningful visual insights using Excel and Power BI.
